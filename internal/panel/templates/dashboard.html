快速上手 / Getting started

1. 添加飞书机器人 / Add a Feishu bot
  1. 飞书群 → 设置 → 群机器人 → 添加「自定义机器人」，复制其 Webhook URL。
  2. 到 飞书机器人 / Feishu Bots 页面新增：填入别名与该 URL。
2. 添加 GitHub Webhook / Add a GitHub webhook
  1. GitHub 仓库 → Settings → Webhooks → Add webhook。
  2. Payload URL 填下方地址；Content type 选什么都支持；Secret 填你在 server.yaml 或对应仓库规则里配置的 secret（若有）。
  3. 选择要监听的事件并保存，再到 仓库规则 / Repos 配置监听与通知目标。
